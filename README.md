# FINAL-YEAR-RESEARCH-PROJECT
Quantization-Induced Accuracy Degradation in Edge-Based YOLOv11-tiny Models for  Post-Disaster Human Detection

## Overview
The primary aim of this research is to evaluate the critical trade-off between model compression (quantization) and 
detection reliability in life-critical disaster scenarios. The study focuses on three core objectives: first, benchmarking 
the baseline performance of YOLOv11-tiny on specialized post-disaster human datasets; second, quantizing the 
model into FP16 and INT8 formats to enable deployment on low-power edge hardware; and third, quantifying the 
"Accuracy Gap" caused by this compression when subjected to simulated environmental stressors like smoke, dust, 
and rubble.
